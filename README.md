## To start

1. Clone this repo

  The best way is to create a new folder with project name, cd into it, and then run

  ```
  git init
  git pull https://github.com/glebmikha/sqlex-pg-ds.git
  ```
2. Run 

```
docker-compose up --build
```
3. Open jupyter
4. Find sqlex.ipynb notebook and open it up.
